# 🌐 Second Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my skills, projects, certifications, resume, and contact information with a clean UI and interactive animations.

## 🚀 Live Demo

🔗 https://cchawla775-art.github.io/Second-portfolio-website/

## 📌 Features

- 🎨 Modern Responsive UI
- 🏠 Hero Section
- 👨 About Me
- 💼 Services Section
- 🛠 Skills Section
- 🏆 Certificate Section
- 📂 Featured Projects
- 📄 Resume Download
- 📧 Contact Section
- 💻 GitHub & LinkedIn Links
- ✨ Custom Animated Cursor
- 🌌 Animated Particles Background
- 💫 Floating Avatar Animation
- 🔥 Mouse Parallax Effect
- ⚡ Loading Screen with Percentage Loader
- 📱 Mobile Friendly Design
- 🌙 Dark Theme Interface

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Particles.js
- Google Fonts (Poppins)
